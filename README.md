Вариант 2. Loop: закольцованный список

Явный интерфейс:

add(h, d)
- добавляет новый узел перед h (то есть новый head)
- возвращает тот самый новый head
- если h == nullptr, то создаёт один узел и возвращает его

insert(h, d)
- добавляет новый узел после узла h
- возвращает указатель на вставленный узел
- если h == nullptr, то создаёт один узел и возвращает его

cut(h) noexcept
- удаляет голову h
- возвращает новый head, ну или же nullptr, если список стал пустым

erase(h) noexcept
- удаляет узел после h
- возвращает h
- если список из одного узла, то ничего не удаляет

clear(h, e) noexcept
- удаляет узлы начиная с h, пока не дойдёт до e (но e не удаляется)
- возвращает e
- при e == nullptr функция удаляет все узлы, затем возвращает nullptr
